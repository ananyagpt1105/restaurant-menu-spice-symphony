## About the Project
This project is a restaurant menu webpage for "Spice Symphony". It displays different food categories including starters, main course, desserts, and beverages. The design uses external CSS for styling and includes images for each menu section.

## Features
- Restaurant landing header
- Food categories with prices
- Responsive layout using Flexbox
- Background image styling
- Section images for menu items

## Technologies Used
- HTML
- CSS

## Concepts Practiced
- External CSS styling
- Flexbox layout
- Google Fonts integration
- Responsive design using media queries
- Background image styling
